# Мои заметки

##  Введение в командную строку  Bash


### Управление Терминалом

Сдвинуть вверх лог выполнения  ранее команд 
**Ctrl-L**

или

```shell
clear
```
```shell
history
```
Выполнить кужную команду из списка History
```shell
!53
```
где 53 - это № команды

Выполнить предыдущую команду

```shell
!!
```

Прервать выполнение запущеной команды

**Ctrl+C**

### Файловые операции

Показать путь текущей директории
```shell
pwd
```

Показать содержание текущего  каталога
```shell
ls
```

Показать подробное содержание указоного  каталога
```shell
ll dir_name
```

Вернуться в домашний каталог текущего пользователя
```shell
cd ~
```

Вернуться обратно
```shell
cd -
```

**/** - знак корня директория

**~** - знак домашнего каталога

Зайти в нужный коталог 
```shell
cd dir_name
```

Показать содержание в виде дерева
```shell
tree
```
Выйти из текущего каталога на 1 шаг вверх

## Linux

Показать красивую ин-фу по системе
```shell
neofetch
```

Показать красивую ин-фу по системе
```shell
inxi -F
```

Показать Диспетчер задач
```shell
htop
```

Показать t CPU/GPU и скорость вентиляторов
```shell
sensors
```

Показать состояние оперативной памяти и подкачки (swap)
```shell
free -h
```

Показать ин-фу о текущем пользователе
```shell
w
```
или
```shell
id
```

Доступные группы
```shell
groups
```
### Софт

Календарь
```shell
cal
```
или на указанный год
```shell
cal 2026
```

Сохранить по **Ctrl+S**, выйти по  **Ctrl+Q**

Запустить python скрипт
```shell
python3 hello.py
```

Скомпилировать код на C++
```shell
g++ main.cpp -o main.bin
```

Запустить бинарный файл
```shell
./main.bin
```

Показать таблицу ASCII
```shell
ascii -d
```

Показать используемые программой библиотеки
```shell
ldd ./main.bin
```

Показать время выполнения скрипта или программы
```shell
time ./main.bin
```

### Сеть

Показать имя компьютера
```shell
hostname
```

Показать ip
```shell
hostname -I
```

Паказать ин-фу по домену
```shell
ping ozon.ru
```

Показать порты
```shell
netstat -an
```
